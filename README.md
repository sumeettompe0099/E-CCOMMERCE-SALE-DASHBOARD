# E-CCOMMERCE-SALE-DASHBOARD

# Ojective:-
Owner of Madhav store wants us to help him in creating an effective dashboard to track and analyze their online sales across india.

# Project Key Points:-
DATA COLLECTION
DATA CLEANING AND WRANGLING
DATA MODELLING
DATA VISUALIZATION
DASHBOARDING DESING
SHARING USEFUL INSIGHTS

# KPI :-

Sum of Amount.

Sum of Profit.

Sum of Quantity.

Sum of AOV(Average Order Value).

# CHARTS:-

1)Sum Of Amount By State.

2)Sum Of Quantity by Payment Mode.

3)Profit by Month.

4)Sum of Amount by Customer Name.

5)Sum Of Quantity by Category.

6)Sum Of Profit by Sub-Category.

#Project Leaning:-
Created Interactive dashboard to track and analyze online sales data.
Used complex parameter to drill down in worksheet and customization using filter and slicers.
created connections,join new table and calculation to manipulate data and enable user driven parameter for visualization.
Used different types of customized visualization (Bar chart,pie chart, Donut chart, Slicers,Scatter chart)etc.

# Situation: 
Fragmented sales data across Indian markets hindered regional profitability analysis and growth strategies.

# Task: 
Engineer an end-to-end BI solution to track Average Order Value (AOV), margins, and sales velocity.

# Action:

Executed ETL & Data Wrangling on disparate tables to ensure 100% accuracy for multi-state reporting.

Implemented Advanced DAX for custom business logic, automating AOV and profitability measures.

Developed dynamic Parameters and Slicers to enable granular stakeholder drill-downs into sub-categories.

Designed interactive Geospatial and Time-Series visualizations to track national performance trends.

# Result:

Reduced manual reporting time by 70% through full dashboard automation.

Identified high-margin sub-categories, enabling strategic inventory reallocation.

Provided a data-backed roadmap for national logistics expansion based on regional revenue hotspots.
